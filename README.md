# Sobre o Projeto

Este repositório foi desenvolvido como parte da disciplina **Aplicações para Dispositivos Móveis (MATC89)**, sob orientação da professora **Laise Cavalcante**. O objetivo é reunir os projetos desenvolvidos ao longo da disciplina, demonstrando de forma prática e estruturada os conceitos abordados em sala de aula.

## Estrutura do Repositório

- **Projeto 1:** `/app-seminario`

## Descrição do Projeto 1

O aplicativo presente na pasta `/app-seminario` tem como finalidade demonstrar, na prática, os conceitos de **gerenciamento de dependências no Flutter** e **acesso a recursos do dispositivo**, incluindo o uso de plugins, permissões do sistema e comunicação com código nativo por meio de Platform Channels.

O projeto foi estruturado para servir como base de apresentação técnica, evidenciando como o Flutter integra bibliotecas externas e interage com funcionalidades do Android e iOS.
